## Descrição

O Task Saga é uma plataforma educacional gamificada inspirada em RPG, desenvolvida para auxiliar professores no gerenciamento de turmas e incentivar alunos por meio de missões, níveis, experiência (XP) e rankings.

Nesta versão foi implementado um fluxo funcional contendo login, visualização das matérias e gerenciamento de turmas.

- Gerenciamento de turmas

- Exclusão de registros de terafa
